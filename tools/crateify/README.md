# IDEAS Crateify

This is a little Rust tool that converts a directory structure with Rust source
files into a valid Rust crate.

## Pre-requisites

Install a Rust toolchain (1.85 or later) (see https://www.rust-lang.org/tools/install)
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Usage

``` bash
crateify <crate_directory>
```
